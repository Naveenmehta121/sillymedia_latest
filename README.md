## 🌟 About The Project

Welcome to the official repository of the **Sillymedia** portfolio website! Sillymedia is a creative agency based in Greater Noida, India, specializing in web development, scrollytelling experiences, and premium interface design.

Our agency's website reflects our commitment to high-end, production-ready design, leveraging modern frameworks to create immersive and interactive digital experiences.

---

## ✨ Key Features

- **Immersive Scrollytelling:** Canvas-based scroll animations and dynamic text overlays that create cinematic user journeys.
- **Micro-interactions:** Custom cursors, variable proximity effects, and intricate Framer Motion transitions.
- **Premium Design System:** Tailored aesthetics emphasizing high contrast, modern typography, and glassmorphism.
- **Responsive Layout:** Pixel-perfect rendering across mobile, tablet, and desktop devices.
- **Performance Optimized:** Fast load times and seamless transitions powered by Vite.

---

## 🛠️ Tech Stack

This project is built with a modern, scalable web stack:

- **Framework:** [React 18](https://reactjs.org/) & [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Backend/DB:** [Supabase](https://supabase.com/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your-username/sillymedia_latest-main.git
   ```
2. Navigate to the project directory
   ```sh
   cd sillymedia_latest-main
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the development server
   ```sh
   npm run dev
   ```
